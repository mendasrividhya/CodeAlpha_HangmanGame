# 🎮 Hangman Game

A modern Hangman Game developed using Python Flask, HTML, CSS, and JavaScript.

## 📌 Features

- Interactive Hangman Game
- SVG Hangman Drawing
- Clickable A-Z Keyboard
- Difficulty Levels (Easy, Medium, Hard)
- Timer
- Score Tracking
- High Score (Saved in Browser)
- Responsive Design
- Modern UI
- Restart Game

## 🛠 Technologies Used

- Python
- Flask
- HTML5
- CSS3
- JavaScript

## ▶️ Run Locally

Install dependencies

```bash
pip install -r requirements.txt
```

Run the application

```bash
python app.py
```

Open your browser

```
http://127.0.0.1:5000
```
